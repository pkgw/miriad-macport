The MIRIAD MacPort Homepage

This is the homepage of the MacPorts package of the MIRIAD radio astronomy suite.

This is not MIRIAD. This is a system that makes it easy to install and update MIRIAD. For more information on MIRIAD, see:

  1. the MIRIAD wiki
  2. the CARMA MIRIAD site
  3. the CARMA bugzilla (which tracks MIRIAD bugs amongst others)
  4. the homepage for the ATNF version of MIRIAD (which this version is not)

So what is the MIRIAD MacPort? MacPorts is a system to ease the installation and management of Unix-oriented software on Macintosh computers. Since MIRIAD is hard to build and many astronomers have Mac laptops on which they want to run it, MacPorts is a useful way to make those people happy.

Current MacPorts version: miriad 4.1.1.20091018 (info)

Installation

Installation is about as easy as we can make it, but still nontrivial. The steps are:

  1. Make sure you will be able to leave your computer running for a few hours. The initial install does not need attention but will take a while to compile everything. (Updates are faster.)
  2. Install MacPorts using these instructions.
  3. Open a terminal. Which telescope will you be reducing data from?
    • The Allen Telescope Array: run sudo port -k install miriad.
    • CARMA: run sudo port -k install miriad +carma.
    • Anything else: contact the port maintainer and ask for a variant to be created for your telescope.
  4. Your computer will spend a long time whirring away and should eventually finish working without giving any error messages. If you see an error message, contact the port maintainer with the exact text of the error.

For general help, see the MacPorts support page, which might have helpful information. That page, however, has no MIRIAD-specific information.

Updating

Once you have MIRIAD installed in this way, you can update your installation with the following steps:

  1. Open a terminal.
  2. Run sudo port sync.
  3. Run sudo port -k upgrade miriad. If nothing happens, you already have the most recent version of MIRIAD that's been pushed to the MacPorts tree.

If you get an error, you might try opening a terminal and running:

  1. sudo port selfupdate
  2. sudo port upgrade outdated

It is also good to run these commands once in a while even if you don't get any errors with MIRIAD. They may, however, take a long time to run.

Updates to the MacPort version of MIRIAD will be pushed out approximately every month. Updates will be released more rapidly for key bugfixes to the tasks or build system — contact the port maintainer to request an update. The tarballs that are pushed are produced by a make dist using the autotools in a pristine CVS tree. (Actually, we've started out by fudging that a bit, but we're going to fix that.)
